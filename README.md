Mình bắt đầu tìm hiểu về lập trình với mong muốn có thể tự tay xây dựng những sản phẩm của riêng mình, từ đó phát triển kỹ năng và tư duy logic.

Ngoài việc học trên lớp, mình còn dành thời gian để:

* Khám phá các công cụ hỗ trợ như Figma, GitHub

Mình luôn mong muốn tạo ra những sản phẩm đơn giản nhưng hữu ích, dễ sử dụng và mang lại trải nghiệm tốt cho người dùng.

# 🌐 Giới thiệu về Website

Đây là một website mình tự thiết kế và xây dựng với mục tiêu mô phỏng một hệ thống giới thiệu sản phẩm và dịch vụ.

Website bao gồm các chức năng chính:

* Trang chủ giới thiệu tổng quan
* Danh sách sản phẩm có thể tìm kiếm và lọc
* Trang chi tiết sản phẩm
* Trang liên hệ

Giao diện được thiết kế theo phong cách hiện đại, tối giản, tập trung vào trải nghiệm người dùng.

# 🌐 Website Project - Trương Anh Phú

## 📌 Giới thiệu

Đây là project website cá nhân được thực hiện nhằm phục vụ mục đích học tập và rèn luyện kỹ năng lập trình web.

## ⚙️ Công nghệ sử dụng

* HTML, CSS, JavaScript
* Thiết kế UI bằng Figma
* Quản lý mã nguồn bằng GitHub

## 📂 Chức năng chính

* Trang chủ (Home)
* Danh sách sản phẩm (List)
* Chi tiết sản phẩm (Detail)
* Trang liên hệ (Contact)

## 🎯 Mục tiêu

* Nâng cao kỹ năng lập trình web
* Hiểu rõ quy trình xây dựng một website hoàn chỉnh
* Tạo ra sản phẩm có thể sử dụng thực tế
